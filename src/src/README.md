# Hospital Management System (Java)

## Description
This is a simple console-based Hospital Management System developed using Java.
It manages patient and doctor details using object-oriented programming concepts.

## Features
- Add and display patient details
- Add and display doctor details
- Uses ArrayList for data storage
- Demonstrates OOP concepts

## Technologies Used
- Java
- GitHub

## How to Run
1. Download all Java files from src folder
2. Compile:
   javac *.java
3. Run:
   java Hospital
