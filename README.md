# Hospital Management System

## Description
This is a Java-based Hospital Management System developed as a console application.
It helps manage basic hospital operations such as patient and doctor records.

## Features
- Add patient details
- View patient details
- Add doctor details
- View doctor details
- Simple menu-driven program

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner class

## How to Run
1. Compile all Java files
2. Run Main.java
3. Follow the menu options

## Purpose
This project is created for academic and learning purposes to understand core Java concepts.
